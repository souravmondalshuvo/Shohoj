FILE RESPONSIBILITIES
═════════════════════

ENTRY
─────
main.js
  • Imports all modules
  • Populates app registry (breaks circular deps)
  • Theme toggle + scroll progress bar
  • DOMContentLoaded boot: initReveal → initCursor → initDotMatrix → initCalculator

CORE  (no DOM, pure logic)
───────────────────────────
core/registry.js
  • Exports a single mutable `app` object
  • main.js writes: app.recalc, app.renderSemesters, app.updateSetupWizard,
    app.runSimulator, app.drawTrendChart, app.buildWhatIfSelect,
    app.importTranscriptPDF, app.applyImport, app.parseTranscriptText
  • Any module that would create a circular import calls app.fn() instead

core/grades.js
  • GRADES map  { 'A': 4.00 … 'F(NT)': null, 'P': null, 'I': null }
  • POINTS_TO_GRADE reverse-lookup array

core/helpers.js
  • detectGrade(val) — GP value → letter grade
  • ordinalSup(n) — 1 → '1st', 2 → '2nd' …
  • getCurrentSeason(), getLastCompletedSemester()
  • countSemesters(), generateSemesterNames()
  • getStartSeason(), getStartYear()
  imports: grades.js, state.js (semesters)

core/departments.js
  • DEPARTMENTS object — 8 keys: CSE EEE BBA ECO ENG ARC PHR LAW
  • Each key: { label, totalCredits, presets[] }
  • Presets drive the setup wizard semester list

core/catalog.js
  • COURSE_CATALOG object keyed by dept
  • 255 courses with name + default credits
  • Powers the autocomplete suggestion portal
  imports: departments.js

core/state.js
  • let semesters = []
  • let whatIfMode = false, whatIfGrades = {}
  • saveState()  — writes to localStorage
  • loadState()  — reads + migrates from localStorage
  • clearState() — resets to empty
  imports: grades.js, departments.js, registry.js (app.recalc, app.renderSemesters)

core/calculator.js
  • calcSemGPA(sem) — single semester GPA
  • recalc()        — full CGPA + all display updates (standing, chart, simulator)
  • autoDetectGrade(semId, cIdx, val, inputEl)
  • onPFChange(semId, cIdx, val)
  imports: grades.js, state.js, helpers.js, registry.js

UI  (DOM manipulation)
───────────────────────
ui/render.js
  • renderSemesters()       — rebuilds entire calculator DOM
  • addSemester(prefill)    — appends new semester row
  • removeSemester(id)
  • addCourse(semId)
  • removeCourse(semId, cIdx)
  • onDeptSelect()          — dept dropdown change handler
  • onStartSemConfirm()     — wizard confirm step
  • usesBestGradePolicy()   — check if dept uses best-grade retake rule
  • getRetakenKeys(sems)    — build retake lookup map
  • getSemCreditWarning(sem)
  imports: grades.js, departments.js, state.js, helpers.js, registry.js

ui/suggestions.js
  • showPortalSuggestions(inputEl, semId, cIdx, matches)
  • onCourseBlur / onCourseInput / onCourseKey — course name input handlers
  • closeSuggestions(id)
  • pickSuggestion(semId, cIdx, fullName, credits)
  imports: catalog.js, state.js, registry.js

ui/charts.js
  • drawTrendChart(canvas, data) — canvas semester GPA trend line
  imports: state.js

ui/simulator.js
  • runSimulator(cgpa, credits, pts) — CGPA goal projections table
  • buildRetakeSuggestions(...)      — ranked retake recommendations
  • buildWhatIfSelect(semId, cIdx, currentGrade)
  • onWhatIfChange(semId, cIdx, grade)
  • toggleWhatIf()
  imports: grades.js, state.js, helpers.js, render.js, registry.js

ui/modals.js
  • exportPDF()                  — jsPDF transcript export
  • getModalTheme()              — dark/light aware modal colours
  • showImportModal(html)        — renders parsed-transcript preview modal
  • hideImportModal()
  • importTranscriptPDF(inputEl) — PDF.js file reader + calls parseTranscriptText
  imports: grades.js, departments.js, state.js, calculator.js, render.js, registry.js

ui/setupWizard.js
  • updateSetupWizard() — updates step indicators (1→2→3)
  • initCalculator()   — first-run init: renderSemesters + recalc
  imports: state.js, helpers.js, registry.js

IMPORT  (transcript parsing)
─────────────────────────────
import/parser.js
  • parseTranscriptText(text)  — main parser (codeOnlyRe, skipNextFrag, all fixes)
  • parseBlobFallback(text)    — fallback for blob extraction

import/importer.js

  imports: grades.js, departments.js, state.js, parser.js, modals.js, registry.js

ANIMATIONS  (no logic, pure visual)

  • Rim glow on proximity, requestAnimationFrame loop


HOW TO DEPLOY
═════════════
1. Delete  js/script.js  from your repo
2. Add all new files maintaining the folder structure above
3. index.html already updated:
     <script type="module" src="js/main.js"></script>
4. No build step needed — ES modules work natively on GitHub Pages
